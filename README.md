# My Final Project: Relationship 911

## Description

"Relationship 911 is a site that allows people to easily access relationship advice and guide ranging from dating, parent/children, marriage advice. It also provides access to well-trained psychologists and counsellors on users' demand"

## Features

- **Advice Categories**: Browse advice for Dating, Parent/Child, and Marriage.
<!-- - **Search & Filter**: Find advice quickly by keyword and category. -->
- **Counselor Directory**: View licensed counselors and request a session.
- **Save Favorites**: Bookmark advice using localStorage.
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Contact Form**: Request a session with a counselor.

## Technologies Used

- Vite
- Vanilla JavaScript
- CSS

## How to Run Locally

1. Clone the repo
2. Run `npm install`
3. Run `npm run start`
4. Open http://localhost:5173
